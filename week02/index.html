Chapter 2: Programming Basics

  • Each statement ends in a new line OR a semi-colon, but it is best practice to have a semi-colon
  • There are seven different data types: String, Symbol (The symbol primitive data type was only introduced in ES6.), Number, Boolean, Undefined, Null
  • typeof outputs the type of the variable, which could be very useful
  • Backslashes \ help us to use characters escaped from the strings
  • Dot operators are extremely useful

Chapter 3: Arrays, Logic, and Loops

  • const myArray = new Array();
  • const myArray = [];
  • delete avengers[3];
  • avengers.pop(); to remove the last item in an array
  • .push adds to the end of the array and .unshift adds to the beginning
  • .slice creates a subarray
  • .sort can sort an array
  • .indexOf cn find if a value is in the array
  • Array.from(setName) creates an array from a set
  • Weak sets can prevent memory leaks

Chapter 4: Functions

  • Functions can contain a block in them that can be run when the function is called
  • Functions can return certain values to be used elsewhere
