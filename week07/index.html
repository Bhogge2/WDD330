Further Functions

  • Functions have properties and methods
  • The call function allow you to call a function and send in a variable
  • You can add properties as well
  • You can call the function inside of itself and then run different code
  • Async Programming allows for tasks to be completed without holding up the program
  • Promises represent the end result of an asynchronous operation

AJAX

  • Fetch API is the standard for requesting and sending data asynchronously accross a network
  • fetch('https://example.com/data') .then( // code that handles the response ) .catch( // code that runs if the server returns an error )
  • redirect() is used to go to another URL
  • response.json() turns the json into a javascript object
  • Http Headers can pass additional information
