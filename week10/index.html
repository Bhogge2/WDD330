Form Validation

  • Form validation is needed to be able to get correct information and give the user direction to do so
  • Form validation protects the developers from things such as SQL injection
  • Key words: required, minlength, maxlength, type, pattern

Fetch

  • The fetch API allows for accessing and manipulating parts of the HTTP pipeline
  • Example: fetch('http://example.com/movies.json') .then(response => response.json()) .then(data => console.log(data));
  • Example: const data = { username: 'example' }; fetch('https://example.com/profile', { method: 'POST', // or 'PUT' headers: { 'Content-Type': 'application/json', }, body: JSON.stringify(data), }) .then(response => response.json()) .then(data => { console.log('Success:', data); }) .catch((error) => { console.error('Error:', error); });
  • Header interface allows for your own header objects
